# MERN Task Manager

A full-stack Task Manager application built using:
- MongoDB
- Express
- React
- Node.js

## Features
- User tasks CRUD
- Admin dashboard
- REST API
- MongoDB database

## How to Run

### Backend
cd backend  
npm install  
nodemon server.js  

### Frontend
cd frontend  
npm install  
npm start
